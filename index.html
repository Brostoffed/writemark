<md-live-editor> live inline editor

Dependency-free markdown web component. The default mode is inline/live: headings, inline formatting, tables, code blocks, task checkboxes, and lists render inside the editor itself while markdown source remains canonical. There is no built-in toolbar; use Markdown shortcuts, slash commands, keyboard shortcuts, or the public action API.

This module demo must be served over http://localhost. Do not open index.html with file://. Run ./serve.sh, double-click start-local.command on macOS, run npm start, or open standalone.html directly.
